Using the wonderful CSS methodology, flexbox, try to move the players into the boxes that match their uniform's colour and symbols.

Once all the players are succesfully contained within the placeholder, you will be congraulted and prompted to move to the next level.

But note! Sometimes the players will be in their boxes, but your CSS syntax might not be perfect, take another look and make sure you've got semicolons everywhere you need them!

In order to apply the styles, type your CSS into either the of the inputs. The 'flex-parent' rules will be applied to the element that contains all the flex items.

While you cannot control which flex-child to add CSS to, the game will ensure you are targetting the correct flex-child when needed.

Not all levels will require you to write CSS for both the flex-parent and the flex-child.
